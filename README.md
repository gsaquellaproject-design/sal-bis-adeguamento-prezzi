# SAL Bis Adeguamento Prezzi

## 🇮🇹 Descrizione

Tool open source per supportare il calcolo dei SAL bis di adeguamento
prezzi negli appalti pubblici italiani. L'obiettivo è ridurre errori e
tempi di elaborazione automatizzando l'applicazione dei prezzari
aggiornati alle lavorazioni già contabilizzate.

## 🇮🇹 Obiettivi principali

- Automatizzare il ricalcolo delle lavorazioni eseguite
- Applicare in modo trasparente prezzari regionali e/o nazionali
- Generare output verificabili e facilmente allegabili ai SAL
- Ridurre l’uso di fogli Excel manuali e formule “nascoste”

## 🇮🇹 Stato del progetto

- Fase: prototipo iniziale / definizione requisiti
- Linguaggio previsto: (Python)
- Focus attuale: modellazione del flusso SAL bis e dei dati in input

---

## 🇬🇧 English Overview

Open-source tool to support automated SAL-bis price adjustment
calculations for Italian public works contracts. The goal is to reduce
manual spreadsheet work and provide auditable, reproducible results.

### Goals

- Automate recalculation of already-executed work items
- Apply updated regional/national price lists
- Produce outputs suitable for official SAL documentation
- Provide transparent, open formulas instead of opaque spreadsheets

### Project status

- Stage: early prototype / requirements definition
- Planned tech stack: (e.g. Python + CLI / simple web UI)
- Current focus: modelling input/output structure and rules
