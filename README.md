# sal-bis-adeguamento-prezzi
SAL bis adeguamento prezzi negli appalti pubblici ita, ai sensi della L. di Bilancio 2026 (L. 199/2025) e del D.L. 50/2022. Il tool automatizza l'applicazione dei prezzari regionali aggiornati alle lavorazioni annotate nel libretto delle misure, riducendo drasticamente i tempi di calcolo manuali per RUP, direttori lavori e imprese appaltatrici.
